🧠 Constructor & Prototype 연습문제

문제 0. Constructor로 객체 만들기

하드코딩으로 만든 학생 객체들이 있습니다:

var 학생1 = { name: 'Kim', age: 20 }
var 학생2 = { name: 'Park', age: 21 }
var 학생3 = { name: 'Lee', age: 22 }

이제는 constructor 문법을 이용해 같은 형태의 객체를 만들고, sayHi()라는 메서드를 추가해 학생1.sayHi() 실행 시 "안녕 나는 Kim이야"가 출력되도록 하세요.

문제 1. 출력 결과 예측

다음 코드의 콘솔 출력 결과는 무엇일까요?

function Parent() {
  this.name = 'Kim';
}
var a = new Parent();

a.__proto__.name = 'Park';
console.log(a.name);

문제 2. 함수가 안 나와요 😢

아래 constructor와 prototype 코드에서 sayHi()가 정상 동작하지 않습니다. 이유를 설명해보세요. 

function Student(이름, 나이){
  this.name = 이름;
  this.age = 나이;
}

Student.prototype.sayHi = () => {
  console.log('안녕 나는 ' + this.name + '이야');
}

var 학생1 = new Student('Kim', 20);
학생1.sayHi();

문제 3. Array 프로토타입 확장

모든 Array에서 사용할 수 있는 remove3()라는 함수를 만들어야 합니다. 이 함수는 배열 내에 있는 숫자 3을 제거합니다. 

var arr = [1, 2, 3];
arr.remove3();

console.log(arr); // 출력: [1, 2]

어떻게 만들어야 위의 코드가 작동할까요? 모든 배열에서 사용 가능하도록 구현 방법을 고민해보세요.
